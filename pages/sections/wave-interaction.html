Wave Interaction

What happens when two waves of the same type overlap?

When two waves of the same type overlap, the new displacement at each point is equal to the displacement from wave one added to the displacement from wave two.

The velocity $v$ for waves of the same type is constant, so an increase in frequency must mean a decrease in wavelength.
